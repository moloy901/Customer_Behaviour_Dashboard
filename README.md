Customer Behavior Data Analytics Project:

Overview:
This project analyzes customer purchasing behavior using Python, SQL, and Power BI. The workflow includes loading the dataset in Python, performing exploratory data analysis (EDA), cleaning and preprocessing the data, running analytical queries in MySQL Workbench, and building an interactive Power BI dashboard.

Tools Used:
Python
MySQL Workbench
Microsoft Power BI

Key Steps:
Load and explore the dataset using Python
Perform EDA and clean the data
Run SQL queries to analyze customer behavior
Build a Power BI dashboard to visualize insights
